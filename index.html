Topological Data Analysis - Finding Structure In Text.

Introduction: Topological Data Analysis (TDA) is a developing branch of data science which uses statistical learning and techniques from algebraic topology, such as persistent homology to study the data. In this project we will be using a range of topological techniques such as persistent homology, and Takens Embedding to determine the shape of each text and then concepts like the Bottleneck distance or the Wasserstein Distance to determine how similair any 2 graphs are. Another interesting problem was converting the text into a form the TDA techniques could more easily work with. TDA systems typically only operate on numerical data and so the sample texts can't be analysed in their original state. We managed to get around this problem by using the python library ntlk to convert the text into a more standard form and then using the python library bag of words to convert the text into a word count vector.

This project hopes to uses topological techniques to compare different texts by the shape of their data and see how effective this is at categorising the different texts..

Background Reading

What is a word count vector?

A word count vector is a convenient way of converting pieces of text to numerical data which works better when using topological techniques. It works by first finding each unique word in the piece of text and then counting the number of times it occurs in each line of the text. This forms a matrix with each line as a row and each unique word as a column. You can then concatenate the data so that each row gets converted into one long row creating a long vector ready for Takens Embedding.

What is algebraic topology?

Algebraic topology uses methods from abstract algebra to answer topological questions by mapping spaces to algebraic objects. It uses abstract algebraic techniques like groups or rings, as many hole structures can be represented by these, to study objects and their properties that are homoeomorphic ie: invariant under continuous deformation.

  1. Homotopy - Two continuous functions from one topological space to another are called homotopic if one can be "continuously deformed" into the other
  2. Homology - Is a general way of associating a sequence of algebraic objects such as abelian groups or modules to other mathematical objects such as topological spaces.
  3. Cohomology - Cohomology can be viewed as a method of assigning richer algebraic invariants to a space than homology.

Homotopy, homology, and cohomology groups are the main areas of study in algebraic topology. In a very broad sense they attempt to capture the number of k-dimensional holes in a topological space.

What is persistent homology?

Persistent homology is a method for computing topological features of a space at different spatial resolutions.More persistent features are detected over a wide range of spatial scales and are deemed more likely to represent true features of the underlying space rather than noise or choice of parameters.

To find the persistent homology of a space, the space must first be represented as a simplicial complex. A distance function on the underlying space corresponds to a filtration of the simplicial complex, that is a nested sequence of increasing subsets.

Simplicial Complexs

Simplicial Complexes are defined as collections of simplices which satisfy the following conditions.

  • The complex must contain every face of any simplex belonging to it.
  • Any simplices that overlap do so on a common face

These simplicial complexes are often used To Triangulate Topological Spaces that they are Homeomorphic to, However in Persistent Homology we use a special type called a Vietoris-Rips Complex. In these complexes two vertices x and y in a dataset are only connected by an edge if for an assigned value e the euclidean distance d(x,y) is less than epsilon

Betti Numbers and Holes

A hole in a mathematical object is a topological structure which prevents the object from being continuously shrunk to a point. When dealing with topological spaces, a disconnectivity is interpreted as a hole in the space.

In algebraic topology, the Betti numbers are used to distinguish topological spaces based on the connectivity of n-dimensional simplicial complexes.

Betti number: Bn = dim(ker(dn))-dim(image(dn+1))

Informally, the kth Betti number refers to the number of k-dimensional holes on a topological surface. The first few Betti numbers have the following definitions for 0-dimensional, 1-dimensional, and 2-dimensional simplicial complexes:

  1. b0 is the number of connected components
  2. b1 is the number of one-dimensional or "circular" holes
  3. b2 is the number of two-dimensional "voids" or "cavities"

Bar Codes and Persistence Diagrams

A bar code is a visual representation of the persistent homology where long bars represent features and short bars represent noise.

A persistence diagram captures the birth and death of holes. The persistence diagram is very similar to the barcode. The x-coordinate of that point is when that point was born. The y-coordinate is when that feature died. The distance between the 45 degree line and the point is the duration of the persistence for that feature. Features that were short lived occur close to the 45 degree line, while more persistent features occur at a greater distance from the 45 degree line.

Takens' Theorem: Let \(M\) be a compact \(C^2\) differentiable manifold of dimension \(m\). Let \(\Phi: M \rightarrow M\) be a \(C^2\) diffeomorphism and let \(\mu: M \rightarrow \mathbb{R}\) be any \(C^2\) map. It is a generic property that the mapping \begin{equation} \Phi_{\phi, \mu}: M \rightarrow \mathbb{R}^{2 m+1}, x \mapsto\left(\mu(x), \mu \phi(x), \mu \phi^{2}(x), \ldots, \mu \phi^{2 m}(x)\right) \end{equation} is an embedding.

Takensâ€™ embedding allows us to convert our vector of data points into a multi-dimensional which can then be analysed and sorted using Persistent Homology. A vector can be reconstructed in phase space where m is the embedding dimension (the smallest dimension required to embed an object) and t is the time delay meaning how big the gap between consecutive points in your new m-dimensional vectors. Thus, instead of analysing the series along time, we analyse its trajectory as a set of visited states in an m-dimensional Euclidean space. This is particularly useful to extract the topological behaviour we are interested in.

In the study of dynamical systems, a delay embedding theorem gives the conditions under which a chaotic dynamical system can be reconstructed from a sequence of observations of the state of a dynamical system. The reconstruction preserves the properties of the dynamical system that do not change under smooth coordinate changes (i.e., diffeomorphisms), but it does not preserve the geometric shape of structures in phase space.

Analysis of Text

Weve talked a lot about TDA in general but there are some issues specific to analysing text based data that we need to adress. We need to find the best way of representing our text as numerical data while still showing the significance of the different structures that exist within the data. the easiest way to convert to numerical data is by using a word count vector.The word count vector is made by making a list of all the unique words that exist within the text, Then, for every line or sentence in the text the instance of each word in the sentence is counted giving a row for each line with a column for each row. This Matrix of results can then be converted into one long vector by the concatenate funtion which can then be analysed by Takens Embedding and then Persistent Homology. While this is one way to analyse text it doesnt capture the full picture of the text only the words and the frequency/patterns in which they appear. To better see what is happening in the text we will use a series of techniques from the NLTK ( Natural Language ToolKit) package in Python.
